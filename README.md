# SmileCare Dental Clinic

Full-stack dental appointment booking system with admin panel.

## Live Demo
[View Live](https://smilecare-gules.vercel.app) · [Admin Panel](https://smilecare-gules.vercel.app/admin/login)

## Tech Stack
React · Tailwind CSS · Node.js · Express · PostgreSQL

## Run Locally
```bash
# Backend
cd backend && npm install && npm run dev

# Frontend  
cd frontend && npm install && npm run dev
```
